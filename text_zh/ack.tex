\section{致谢}

本工作的很大一部分基础来自他人的成果。Web3 基金会研究团队，尤其是 Alistair Stewart 和 Jeff Burdges，负责开发了 Polkadot 的安全机制 \textsc{Elves}，它使得 \Jam 的内核计算成为可能。同一团队还负责了 Sassafras、\textsc{Grandpa} 和 \textsc{Beefy} 的设计。

Safrole 是 Sassafras 的一种简化版本，它是在 Davide Galassi 与 Alistair Stewart 的仔细审阅下完成的。

最初的 CoreJam \textsc{rfc} 在 Bastian Köcher 与 Robert Habermeier 的审阅下得以完善，该提案中的大多数关键要素最终被纳入了本工作。

\textsc{pvm} 是 \emph{PolkaVM} 软件原型的部分简化形式的形式化版本，该原型由 Jan Bujak 开发。Cyrill Leutwiler 对本文所报道的 \textsc{pvm} 实证分析作出了贡献。

\emph{PolkaJam} 团队，尤其是 Arkadiy Paronyan、Emeric Chevalier 与 Dave Emett，在 \Jam 协议底层方面的设计中发挥了关键作用，特别是在默克尔化与 \textsc{i/o} 相关的部分。

自论文发布以来，众多代码仓库的贡献者帮助修正了错误。在此一并致谢。

当然，还要特别感谢令人敬佩的 Lemon Jelly，即 Fred Deakin 和 Nick Franglen，感谢他们创作的三张最美妙的专辑之一，其首张专辑的封面艺术也为本文的背景艺术提供了灵感。
